# Tampermonkey (油猴的入门与AI结合指南)

## 什么是 Tampermonkey (油猴)？

Tampermonkey，通常被国内用户称为“油猴”，是世界上最流行的浏览器扩展，也是一个用于管理和运行 UserScript（用户脚本）的管理器。
通过它，我们可以往任意第三方网站强行注入一段自定义的 JavaScript 和 CSS 代码，以此来实现：
- 隐藏网页中的广告
- 提取受保护或隐藏的页面数据，制作自动化表格
- 自动化繁琐的操作流程，辅助日常工作流

对于内部工具编写而言，**它比制作完整的 Chrome 扩展要轻量非常多，且更新下发极快**。您只需把一个 `.user.js` 地址丢给需求方，他点击一下就能无感安装。

### 常用平台与资源

* **[Tampermonkey 官方网站](https://www.tampermonkey.net/)**: 提供各浏览器版本的插件安装及官方指南。
* **[官方脚本资源库](https://www.tampermonkey.net/scripts.php)**: 官方维护的一些辅助演示。
* **[GreasyFork (油叉)](https://greasyfork.org)**: 世界上最活跃、最大规模的用户脚本库，您可以在这搜索到全网开发者为各大网站编写的功能增强脚本，极具参考价值。

## 为什么选油猴？与直接开发 Chrome 扩展的横向对比

在内部做效率提升工具时，经常会面临“我们要不要开发个 Chrome 插件？”的讨论，下面图表是两者的主要抉择考量：

| 维度 | Tampermonkey 油猴脚本 | 标准 Chrome Extensions (浏览器扩展) |
| --- | --- | --- |
| **开发难度** | **极低**。单个纯文本 `.js` 文件即可开工，基于纯 JavaScript/CSS 极简语法。 | **较高**。需熟悉 Manifest 规范、Background / Content / Popup 的通信隔离机制与打包流程。 |
| **发行与审核周期** | **无周期，秒级**。员工直接将链接或代码发到群里即可复制使用，0 等待。 | **漫长**。上架 Chrome Web Store 需要开发者账号、数天的审核，内网发行也需要经历繁杂的打包分发流程。 |
| **迭代更新** | 支持在自己的内网服务器配置 Check URL，自动在用户端无缝同步最新版本。 | 需通过复杂的重新发版才能覆盖全员应用，极难控制碎片化版本。 |
| **功能权限 (Capabilities)** | 能对页面 DOM 深度改造，支持规避浏览器跨域限制发送特定 Request。 | 具备统治级权限。能劫持网络请求，干涉浏览器历史记录、书签及所有底层系统级别交互。 |
| **适用场景** | 测试验证原型理念（PoC）；提取表单结构；增加几个便捷内部按钮。 | 涉及多窗口管理的大型工具、强交互体系或严肃商业化。 |

## 如何安装和使用？

### 步骤一：安装浏览器扩展
1. 访问 Chrome Web Store (或 Edge Add-ons)。
2. 搜索并安装 **Tampermonkey** 官方组件。
3. 安装完毕后，在浏览器右上角的拼图图标里将油猴图标“固定”，方便后续管理。

### 步骤二：安装您的第一个脚本
我们在本实验室专门准备了一个体验演示脚本当作练习。
前往 [💡 试验场 - 油猴脚本靶场测试](../playground/tampermonkey-test.md)，点击页面内的安装地址，此时油猴会自动拦截并弹出脚本源码供您点击“安装”。

## 脚本的基础结构元数据解析

不管是手写还是 AI 生成的油猴脚本，其开头都有一个特征明显的元数据块（Metadata Block）。熟悉它可以避免 80% 的使用坑。

```javascript
// ==UserScript==
// @name         脚本的名字 (用户看见的名字)
// @namespace    http://tampermonkey.net/
// @version      1.0
// @description  干啥用的？写给使用者看
// @author       作者
// @match        https://*.iyunquna.com/*  <-- 最重要的参数之一，定义这个脚本在什么网站上才会高亮运行
// @grant        GM_xmlhttpRequest        <-- 非常强大的功能，允许突破跨域请求，获取接口
// ==/UserScript==

(function() {
    'use strict';
    // 你的业务代码就写在这里！由于是被注入到目标页面，所以你可以为所欲为。
})();
```

## 配合大语言模型 (AI Tools) 极速编写脚本

对于现代开发者或运营人员，你可以完全无需手搓所有 JS 代码。利用如下的话术配合大语言模型（如 ChatGPT / Claude），几秒就能搭出属于自己的第一个生产力脚本工具：

> **💡 AI Prompt (指令模版)**
> 
> "你好，请帮我生成一个 Tampermonkey（油猴）脚本。
> 我希望在 `https://example.com/reports` 这个页面中，能够自动寻找 ID 为 `export-table` 的这个按钮，如果是不可用的状态，则移除它的 `disabled` 属性，并且给它加上鲜艳的红色背景。
> 请给出完整的 UserScript 头部约束（只需要 @match 在目标域名，不需要任何特殊 @grant），然后用原生 JS 实现核心逻辑，代码加上防止未加载完毕报错的监听控制。"

通过类似上述对话，你会非常快地拿到一份结构完整且自带元数据块的代码。接着便可以在这实验室里通过 GitHub 的 Raw 地址发布，或让内部人员复制使用。
