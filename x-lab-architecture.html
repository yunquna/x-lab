YQNUS X-Lab 总体架构

VitePress · Chrome Extensions · CI/CD · Tampermonkey · GitHub Pages

技术栈

  • VitePress ^1.2.2 — 文档引擎
  • Vue 3 — SPA 运行时
  • Node.js 18+ — 构建环境
  • Chrome Extension MV3 — 插件框架
  • Tampermonkey — 用户脚本引擎
  • GitHub Actions — CI/CD流水线

部署流程

  • 推送 main 分支 → 触发 Actions
  • actions/checkout → setup-node
  • npm ci → npm run docs:build
  • 上传 dist 产物至 Pages
  • 2分钟内自动上线生效
  • 用户刷新即可获取最新版本

内容模型

  • 教程: Markdown + PPT + 录屏
  • 工具: Chrome扩展源码归档
  • 试验场: 在线编辑器+测试靶场
  • 脚本: public/scripts 直接引用
  • 全中文 UI 与文档
  • 面向非研发人员的自助平台
